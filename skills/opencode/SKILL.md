# OpenCode-Specific Skills

Skills optimized for OpenCode with multi-session and tmux.

## Available Skills

### @opencode/multi-session
Manage multiple concurrent sessions.

- Session coordination
- State sync
- Cross-session context

### @opencode/tmux-orchestrate
Orchestrate tmux sessions for parallel workflows.

- Window management
- Pane orchestration
- Session templates

### @opencode/agent-council
Run multi-agent Council meetings.

- Meeting coordination
- Consensus building
- Decision aggregation

## Usage

```bash
# Install OpenCode-specific skill
omg skills install @opencode/multi-session

# Use in conversation
@opencode/tmux-orchestrate create dev environment
```
