# Claude-Specific Skills

Skills optimized for Claude Code with extended thinking and artifacts.

- Chain-of-thought reasoning
- Step-by-step decomposition
- Self-verification

### @claude/artifacts
Create interactive artifacts for UI/UX prototyping.

- Server discovery
- Resource coordination
- Tool chaining

```bash

omg skills install @claude/extended-reasoning
```
