# {{ project_name }}

A powerful command-line tool built with Commander.js.

## Installation

```bash
npm install
npm link  # Optional: install globally
```

## Usage

```bash
# Show help
{{ project_name }} --help

# Show version
{{ project_name }} --version

# Greet someone
{{ project_name }} greet World
{{ project_name }} greet World --loud

# Display system info
{{ project_name }} info

# Echo a message
{{ project_name }} echo "Hello!"
{{ project_name }} echo "Repeat me" --repeat 3
```

## Commands

- `greet <name>` - Greet someone by name
- `info` - Display system information
- `echo <message>` - Echo a message back
