## Implementation Details

### record_stop_block() Logic
1. Create new state with ts, count=1, session_id, reason
2. If tracker exists and elapsed < window:
   - Increment count from old state
   - Preserve session_id from old if not provided
   - Preserve reason from old if reason="unknown"
3. Write to tracker atomically

### is_stop_block_loop() Logic
1. Load tracker state
2. Extract tracker_session_id = state.get("session_id", "")
3. If tracker_session_id AND current session_id AND they differ → return False
4. Otherwise: check elapsed < window AND count >= threshold
