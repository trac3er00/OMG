## Dedup Logic Summary
- File: `.omg/state/.last-stuck-ts` (contains float timestamp)
- Reads timestamp, compares with time.time()
- If delta < 60s → skip injection (return without @stuck)
- If delta >= 60s or file missing/corrupt → inject + write new timestamp
- Failure threshold: requires `active` dict to be non-empty (≥2 failures tracked)
