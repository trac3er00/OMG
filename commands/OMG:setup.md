# /OMG:setup — DEPRECATED

This command has been merged into `/OMG:init`.

Run `/OMG:init setup` for the native OMG adoption wizard.

```
/OMG:init setup                          # Same as old /OMG:setup
/OMG:init setup --preset balanced        # Direct preset selection
/OMG:init setup --non-interactive        # Non-interactive mode
/OMG:init                                # Auto-detect: project setup, domain init, or health check
```
