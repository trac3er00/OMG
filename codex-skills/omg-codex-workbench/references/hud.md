# OMG Codex HUD

- The Codex HUD is a standalone OMG workbench command, not a native statusline hook.
- Primary command: `omg-codex-hud --project "$PWD"`.
- Use `--json` when Codex should parse fields before acting.
- Expect fields for project mode, handoff presence, notes presence, git state, installed OMG Codex skills, and provider availability.
