# Migration Guide: v2.5 → v2.6

## Overview

v2.6 introduces the AI Work OS front door: `omg "<goal>"` as the primary entry point.

## Breaking Changes

None. All existing commands continue to work.

## New Features

### Single Front Door

**Before (v2.5):**

```bash
npx omg instant "make a landing page"
```

**After (v2.6):**

```bash
npx omg "make a landing page"
```

## Deprecated (still works, shows warning)

- `omg instant "<goal>"` → use `omg "<goal>"` instead

## Pack Structure Changes

- New unified `packs/` directory with all packs
- Old `packs/goals/` and `packs/domains/` still work (backward compat)

## New Systems

- TaskClassifier: automatic goal classification
- ProofScore: 5-dimension verification scoring
- Wow Flows: 5 production-ready project templates
