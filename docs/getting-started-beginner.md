# Getting Started for Beginners

OMG helps you get useful output fast without needing a complex setup flow in your head.

## Start in 3 steps

1. Check your machine: `npx omg env doctor`
2. Preview setup safely: `npx omg install --plan`
3. Apply when ready: `npx omg install --apply`

## Your first win

Try one simple command:

```bash
npx omg instant "make a landing page"
```

## What to expect

- OMG helps generate working project scaffolds quickly.
- Governance is on by default, so risky changes are easier to spot.
- Proof and status surfaces help you see what happened.

## Good beginner habits

- Start with short prompts.
- Review the preview before applying setup.
- Use `npx omg proof open --html` when you want a clearer verification view.
